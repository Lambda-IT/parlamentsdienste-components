# pd-list

## Usage

#### Simple list

```html
<pd-list>
    <pd-list-item>Text</pd-list-item>
    <pd-list-item>Text</pd-list-item>
    <pd-list-item>Text</pd-list-item>
</pd-list>
```

#### Status list

```html
<pd-list>
    <pd-list-item status="unset">Text</pd-list-item>
    <pd-list-item status="success">Text</pd-list-item>
    <pd-list-item status="danger">Text</pd-list-item>
</pd-list>
```

## Styles

| Style                    | Default       | Description                        |
| ------------------------ | ------------- | ---------------------------------- |
| `--pd-list-border-color` | `gray-3`      | bottom border color of a list item |
| `--pd-list-background`   | `transparent` | background color of a list item    |
